# **App Name**: HashMyLinks

## Core Features:

- Profile Creation: Users create profiles with a custom URL that will represent their unique entrypoint.
- Link Addition: Users add links to their profile, each with a title and associated URL. The application uses URL-based storage, which avoids a back-end.
- Profile Preview: Allow users to preview their profile as it would appear to visitors.
- Customization Options: Users can select basic theme options to personalize the appearance of their page, like colors or fonts.
- AI-Driven Theme Suggestions: Uses a generative AI tool to suggest theme customizations based on user input like the type of content they're sharing.

## Style Guidelines:

- Primary color: Teal (#008080) to evoke a sense of sophistication and trustworthiness.
- Background color: Off-white (#F5F5DC) to provide a clean and readable surface.
- Accent color: Coral (#FF7F50) to draw attention to important elements and create visual interest.
- Body and headline font: 'Montserrat', a versatile sans-serif known for its modern and clean appearance.
- Outline style icons for a modern, minimalist aesthetic.
- Mobile-first, single-column layout for optimal viewing on all devices.
- Gentle fade-in animations for links as the page loads to add a touch of elegance.
